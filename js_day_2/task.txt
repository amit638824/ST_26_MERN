TOPIC 1: BASICS — console, alert, prompt, confirm
================================================================================

Task 01

Task 02

Task 03
Write a program that asks the user's name using prompt() and displays it using console.log().
(No number conversion needed — name is text.)

Task 04
Write a program that asks the user's city using prompt() and displays it using document.write().

Task 05

TOPIC 6: SWITCH STATEMENT
================================================================================

Task 30 — Day of the Week
  - Use a predefined variable dayNumber = 4.
  - Use a switch statement to display the corresponding day of the week.
  - Example: 1 → Monday, 2 → Tuesday, ..., 7 → Sunday.

Task 31 — Simple Calculator
  - Use predefined values num1 = 50, num2 = 10, and operator = "-".
  - Use a switch statement to perform the selected operation and display the result.

Task 32 — Month Name
  - Use a predefined variable monthNumber = 8.
  - Use a switch statement to display the corresponding month name.
  - Example: 1 → January, 12 → December.

Task 33 — Grade Message
  - Ask the user for a grade (A, B, C, D, or F) using prompt().
  - Use a switch statement to display an appropriate message.
  - Example:
      A → Excellent
      B → Very Good
      C → Good
      D → Needs Improvement
      F → Fail
  (No number conversion needed — grade is text.)

Task 34 — Traffic Light
  - Ask the user for a traffic light color (red, yellow, or green) using prompt().
  - Use a switch statement to display the correct action.
  - Example:
      red    → Stop
      yellow → Wait
      green  → Go
      Any other input → Invalid Color
  (No number conversion needed — color is text.)
