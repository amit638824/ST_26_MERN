TOPIC 1: ARRAY map() METHOD — Transform Every Element
================================================================================

Task 01
Create an array nums = [1, 2, 3, 4, 5]. Use map() to multiply every element by 2
and print the new array.

Task 02
Create an array nums = [5, 10, 15]. Use map() to add 10 to every number and print
the new array.

Task 03
Create an array names = ["rohan", "rahul", "sneha"]. Use map() to convert every name
into uppercase and print the new array.

Task 04
Create an array nums = [2, 3, 4]. Use map() to calculate the square of every number
and print the new array.
(Hint: 2 × 2 = 4)

Task 05
Create an array nums = [2, 3, 4]. Use map() to calculate the cube of every number
and print the new array.
(Hint: 2 × 2 × 2 = 8)

Task 06
Create an array prices = [100, 200, 300]. Use map() to convert every price into a
string with the ₹ symbol and print the new array.
(Example: "₹100")

Task 07
Create an array words = ["apple", "mango", "kiwi"]. Use map() to find the length of
every string and print the new array.

Task 08
Create an array names = ["HELLO", "WORLD"]. Use map() to convert every string into
lowercase and print the new array.

Task 09
Create an array names = ["Rohan", "Rahul"]. Use map() to add the word "Student -"
before every name and print the new array.
(Example: "Student - Rohan")

Task 10
Create an array nums = [10, 20, 30]. Use map() to multiply every element by its index
and print the new array.
(Hint: index starts from 0)

TOPIC 2: ARRAY filter() METHOD — Select Elements by Condition
================================================================================

Task 11
Create an array nums = [1, 2, 3, 4, 5, 6, 7, 8]. Use filter() to print only even
numbers.

Task 12
Create an array nums = [1, 2, 3, 4, 5, 6, 7, 8]. Use filter() to print only odd
numbers.

Task 13
Create an array nums = [10, 30, 55, 70, 90]. Use filter() to print only numbers
greater than 50.

Task 14
Create an array nums = [5, 10, 15, 25, 30]. Use filter() to print only numbers less
than 20.

Task 15
Create an array nums = [-5, 10, -2, 8]. Use filter() to print only positive numbers.

Task 16
Create an array nums = [-5, 10, -2, 8]. Use filter() to remove negative numbers and
print the new array.

Task 17
Create an array names = ["rohan", "rahul", "sneha", "om"]. Use filter() to print only
names whose length is greater than 5.

Task 18
Create an array nums = [10, 15, 20, 25, 33]. Use filter() to print only numbers
divisible by 5.

Task 19
Create an array nums = [1, 2, 2, 3, 3, 4]. Use filter() to remove duplicate values
and print the new array.
(Hint: use indexOf())

Task 20
Create an array marks = [20, 35, 40, 50, 60]. Use filter() to print only marks greater
than or equal to 35.

Task 22
Create an array colors = ["red", "blue"]. Use push() to add "green", "yellow", and

Task 23
Create an array subjects = ["Math", "English"]. Use push() to add your favorite subject

Task 24
Create an array nums = [1, 2, 3]. Use push() to add 4 and print the updated array.

Task 28
