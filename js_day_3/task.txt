TOPIC 1: BASIC FUNCTIONS — First Simple Functions
================================================================================

Task 01
Write a function that prints "Hello World" using console.log().

Task 02
Write a function that prints your name.
(Example: console.log("Rahul");)

Task 03
Write a function using num1 = 10 and num2 = 5. Add them and print the result.

Task 04
Write a function using num1 = 20 and num2 = 8. Subtract them and print the result.

Task 05
Write a function using num1 = 6 and num2 = 7. Multiply them and print the result.


================================================================================
TOPIC 2: INBUILT FUNCTIONS — alert, prompt, confirm, parseInt, Math
================================================================================

Task 06
Use alert() to display "Welcome to JavaScript".

Task 07
Use prompt() to take the user's name and print it using console.log().
(No number conversion needed — name is text.)

Task 08
Use confirm() to ask "Do you want to continue?" and print the result using console.log().
(Result will be true or false.)

Task 09
Use parseInt("25") to convert the string into a number and print it using console.log().

Task 10
Use Math.max(10, 20) to find the larger number and print the result using console.log().

TOPIC 3: USER DEFINED FUNCTIONS — Create Your Own Functions
================================================================================

Task 13

(Hint: 3 × 3 × 3 = 27)

Task 14

Task 15

================================================================================
TOPIC 4: NAMED FUNCTIONS — Functions with a Name
================================================================================

Task 16
Create a named function greet() that prints "Good Morning".

Task 17
Create a named function showAge() that prints your age.
(Example: console.log(20);)

Task 18
Create a named function findArea() using length = 10 and width = 5.
Calculate and print the area of a rectangle.
(Hint: area = length × width)

Task 19
Create a named function findPerimeter() using length = 10 and width = 5.
Calculate and print the perimeter of a rectangle.
(Hint: perimeter = 2 × (length + width))

Task 20
Create a named function isPositive() using num = 12.
Check whether the number is positive and print the result.
